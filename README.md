# Kiosco Stock (MVP) - React + Vite (JS) + Node 20

Sistema simple de control de stock **sin backend**:
- Productos (CRUD)
- Movimientos: IN / OUT / ADJUST
- Stock calculado desde movimientos
- Alertas de stock bajo
- Persistencia en localStorage
- Tests unitarios (Vitest) para dominio

- Node 20+
